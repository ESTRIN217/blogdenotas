📝 Blog de Notas Pro (Material You)
Un editor de notas moderno para Android, diseñado bajo los principios de Material Design 3 (Material You) e inspirado en la simplicidad y potencia de Google Keep.

✨ Características Principales
🎨 Diseño Dinámico: Interfaz adaptable con colores dinámicos que se ajustan al fondo de pantalla del usuario (Material You).

📂 Almacenamiento Seguro (SAF): Implementa el Storage Access Framework de Android para una gestión de archivos moderna, segura y privada.

🖊️ Editor Enriquecido: Soporte para texto plano con capacidades de personalización de fondo y estilo.

📎 Adjuntos Multimedia: Potencia tus notas agregando:

🎙️ Audio: Grabaciones directas en la nota.

🖼️ Imágenes: Galería y fotos.

🎨 Dibujos: Lienzo integrado para bocetos rápidos.

🫧 Burbuja Flotante: Servicio flotante (FloatingService) para tomar notas rápidas sin salir de otras aplicaciones.

🛠️ Tecnologías Utilizadas
Lenguaje: Java ☕

Arquitectura: Android Jetpack & Material Components (M3).

UI/UX: XML con soporte para modo oscuro y temas dinámicos.

Asistencia: Desarrollado con el apoyo de Gemini para optimización de código y lógica avanzada.

🚀 Cómo funciona
Persistencia: Las notas se guardan en formato .txt con metadatos HTML para preservar los colores de fondo.

Multitarea: Puedes minimizar el editor a una burbuja flotante para mantener la productividad.

Compatibilidad: Diseñado para funcionar perfectamente en versiones modernas de Android (SDK 31+ para colores dinámicos).
